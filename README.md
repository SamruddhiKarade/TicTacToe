# TicTacToe
A web-based Tic-Tac-Toe game built with HTML, CSS, and JavaScript featuring dynamic gameplay, winner detection, and a user-friendly interface.

---

## Project Overview
This project demonstrates the implementation of game logic, DOM manipulation, and basic user interaction using core web technologies.

---

## Objectives
- To build a functional two-player Tic-Tac-Toe game
- To practice JavaScript logic and event handling
- To design a clean and user-friendly interface

---

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)

---

## Features
- Two-player gameplay
- Player name input
- Turn-based system
- Winner detection
- Reset and replay options
- Simple and clean UI

---

## Project Structure
index.html → Home page
game.html → Game board
result.html → Result page
styles.css → Styling
game.js → Game logic
home.js → Home page logic
result.js → Result handling

---

## How to Run the Project
1. Download or clone the repository
2. Open `index.html` in any modern web browser
